Язык: Python
Библиотеки: BeautifulSoup4, requests, asyncio, json, time, csv, re, os


scr-ozon.py -- синхронный простой скрепер озона
asy-scr-ozon.oy -- асинхронный скрепер озона

Фотографии собираются в папку img в папках, названных по имени товара
Выходные данные собираются в ozon_res.csv
Поля ozon_res.csv: "Наименование", "Цена со скидкой",
                   "Цена без скидки", "Описание",
                   "Ссылки на картинки", "Имена картинок",
                   "Разделы", "Ссылка на товар"
В случае ошибок или отсутствия информации поле заполняется 'no_data'
Недочеты: не решена проблема блокировки ip озоном при запросах на https://www.ozon.ru/..., 
          при запросах на другие домены озона(картинки, json-ы) проблем нет
